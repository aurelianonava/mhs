Project 1 - Version 3

As you can see, the point is visible, but barely. We will now make the point more visible by increasing the stroke weight using the strokeWeight() function.
Go to the p5.js Resource Page at: p5js.org

  1. Navigate to Reference

  2. Got to Shape and select the strokeWight() function to read the documentation on how to use it. Knowing how to access the documentation for programming tools is very important.

  3. Apply the strokeWeight() function and use the integer 3 for the argument. This should increase the thickness of your dot.
