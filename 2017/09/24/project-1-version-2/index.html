Project 1 - Version 2

Now we will add the \ element that will allow us to connect to the p5.js visual library. P5 is a class, and classes have things called functions [^1] ( tools ) that we can use and these will allow us to start drawing in the \ element.

Between head element inset:

<script data-require="p5.js@*" data-semver="0.3.0" src="https://rawgit.com/lmccart/p5.js/0.3.0/lib/p5.js"></script>

<script data-require="p5.dom.js@*" data-semver="0.3.0" src="https://rawgit.com/lmccart/p5.js/0.3.0/lib/addons/p5.dom.js"></script>

Between the body element insert: 

<script>
       function setup() {
           canvas = createCanvas(480, 120);
           background(200);
       }

       function draw() {

           point(240, 60);
       }
</script>

Note:

If createCanvas(); does not have values for the two arguments, their default values are (100, 100).

Final code:

<script data-require="p5.dom.js@*" data-semver="0.3.0" src="https://rawgit.com/lmccart/p5.js/0.3.0/lib/addons/p5.dom.js"></script>
