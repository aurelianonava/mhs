Spring 2018: Instructions for Learning Programming With p5.js.

I should have spoken to you already and proposed to you what you will do this semester. Select a video that covers programming concepts you find interesting, watch the video, carefully take notes with OneNote, run the code in the video on link JSBin, and put together a presentation using Powerpoint, Google Slides, or any other of your choice. As an alternative to the presentation, you can choose to write ( use this writing template ) a brief paper on concepts that caught your attention having watched the video.

Take a look at the rest of the videos for The Coding Train.

What is p5.js? What can you do with it? Where did it come from? What is its relationship with Processing? p5.js is both a library for JavaScript and a beginner-friendly editor where you can build creative computational projects in the browser. It can be used for art, data visualization, and websites alongside other tools like HTML5, CSS and JavaScript.

A few examples of the videos you can choose:

1.1: Introduction - p5.js Tutorial

1.3: Basics of drawing - p5.js Tutorial

In this video, you will learnt about the basics of drawing shapes in a web browser using the p5.js library.

1.4: Color - p5.js Tutorial

In this video, you will learnt how to add colors to your p5.js sketches.

2.1: Variables in p5.js (mouseX, mouseY) - p5.js Tutorial

In this video, I introduce the concept of variables in p5.js. The setup() and draw() functions are covered as well as mouseX and mouseY.

2.2: Variables in p5.js (Make your own) - p5.js Tutorial

In this video, you will look at how to create your own variables and how to use them to animate elements of your p5.js sketches.

  1. Introduction ( what is the video selected about )
    Project name
    Project URL ( to Youtube p5.js video )
    Project JSBin URL ( the code for finished p5.js selected video )

  2. Explanation of the selected video project
