In the case of p5.js, fucntion draw() continues to repeat as long as the test in if(), what is inside the parentheses, continues to pass–continues to be true.

What will happen here? The point will move along the x axis until it reaches x = 240, which makes that test false ( does not pass: 240 > 240 is NOT true. ), therefore the statement inside the { } curly braces executes ( runs ) and x is repositioned at x = 0.

Try it out.

[See me here]

So, again, when the if()test is false, the function draw(); continues to runs. When the if()test is true, the code inside the { }curly braces runs. The computer determines whether the test is true or false by evaluating the expression inside the parentheses.

We have at present an if() condition that loops the point for a reason. It is important for you to understand what makes this effect. Note: Each time x is reset back to 0, the test inside if() continues to be false ( not pass ), therefore the function draw runs over and over again, executing if() over and over again. The test in if() becomes true ( passes ) only for an instance, when it reaches 241, and when it does, it allows the statement inside the { } curly braces to run, resenting x to x = 0, which in turn makes the point move again until it reaches 241 again, making the if() test pass again, and again.

If() rule:

if (condition) {
block of code to be executed if the condition is true
}

The else statement:

if (condition) {
block of code to be executed if the condition is true
} else {
block of code to be executed if the condition is false

      • Common loops
        • Start at 0 and count up to 9
          • for (var i = 0; i < 10; i = i + 1){}

          • for (var i = 0; i < 101; i = i + 10){}
        • Start at 100 and count down to 0 by 5
          • for (var i = 100; i >= 0; i = i - 5){}
      • Initialization (first element of for loop) - A variable is declared and initialized for use within the body of the loop. This variable is most often used inside the loop as a counter.
      • Boolean test (middle element of for loop) - The same as the boolean tests found in conditional statements and while loops. It can be any expression that evaluates to true or false.
      • Iteration expression (last element of for loop) - An instruction you want to happen with each loop cycle. It is executed at the end of each cycle through the loop.
      • for (var i = 0; i < 10; i = i + 1){} Declare a variable i, while i is less than 10 repeat this code, at the end of each iteration add 1 to i.
